Machine Learning in Time Series

Professor Marcelo Villena Ch., PhD

Course Description

This course introduces the fundamentals and advanced techniques of Machine Learning applied to the analysis and modeling of Time Series. Both the theoretical foundations and the practical implementation of the models are covered.
