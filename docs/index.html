Machine Learning in Time Series

Prof. Marcelo Villena Ch., PhD

Course Description

This course introduces the fundamentals and advanced techniques of Machine Learning applied to the analysis and modeling of Time Series. Both the theoretical foundations and the practical implementation of the models are covered.

Contents

# Lecture Topics
1 Lecture 1 Historical Evolution, the Prediction Problem, and Shrinkage
2 Lecture 2 Kernel Methods and Support Vector Machines (SVM) for Time Series
3 Lecture 3 High-Dimensional Linear Dynamics (BVAR, Sparse VAR, & Structured Regularization)
4 Lecture 4 Deep Learning for Sequences (RNN/LSTM and Attention-Based Forecasting)
5 Lecture 5 Trees, Boosting, and Ensembles for Nonlinear Forecasting
6 Lecture 6 The Frontier: Attention, Text, and Causal ML

Prerequisites

  • Basic knowledge of statistics and probability
  • Programming in Python or R
  • Linear algebra
